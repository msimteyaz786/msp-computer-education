MICROSOFT WORD
All these questions of microsoft word are the most important for interview or placement.

Microsoft Word is a word processing software package that is used to type letters, reports and other documents.
MS Word comes in version some of the old version are MS Word 2003/2002 or Word 97 etc.

→ Its purpose is to allow users to type and save documents.
→ Similar to other word processors, it has helpful tools to make documents
→ Spelling and Grammer checker, word count(this also counts letters and lines)
→ Insert Pictures in documents.
→ Choice of typefaces.
→ Spacial codes.
→ Tables, Web Pages, Graphs, etc.
→ Display synonyms of words and can read out the text.
→ Prints in different ways.
